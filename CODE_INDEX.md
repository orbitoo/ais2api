## Repository Summary

- The repository is a browser-backed API proxy for Google AI Studio / Generative Language APIs.
- Node.js owns the public HTTP API, admin UI, account rotation, and the browser/WebSocket orchestration.
- A browser-side script performs the actual `fetch` requests from inside the logged-in page context.
- Helper scripts cover auth capture, auth archive packaging, archive extraction tests, and encrypted log decoding.

## Runtime Flow

1. `unified-server.js` loads config, auth sources, and starts HTTP + WebSocket servers.
2. `BrowserManager` launches Camoufox/Firefox with a chosen storage state and injects `black-browser.js` into the target page.
3. Incoming API calls are translated into browser-side proxy requests and forwarded over WebSocket.
4. `black-browser.js` performs authenticated `fetch` calls from the page context and streams chunks back to Node.js.
5. The server returns Google-native responses or adapts them to the OpenAI chat-completions format.
6. Helper scripts maintain auth material, packaged archives, and encrypted log inspection.
