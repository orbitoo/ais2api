# 🎯 Repository Agent Notes

## 🌏 语言约定

1. 与用户交流时，使用简体中文。
2. 代码、脚本、注释默认使用英文。

## 📚 代码索引

1. 仓库内提供自动生成的代码索引文件：`CODE_INDEX.md`
2. 在阅读代码前，优先先看 `CODE_INDEX.md`，快速了解：
   - 仓库总览
   - 运行流程
   - 文件职责
   - 关键类 / 函数 / 路由 / 环境变量
3. 当代码结构可能已变化时，不要盲信旧索引，应先刷新索引再继续分析。

## 🔄 更新代码索引

优先使用以下命令更新索引：

```bash
npm run update:code-index
```

它会执行：

```bash
node scripts/update-code-index.js
```

并重新生成：

```bash
CODE_INDEX.md
```

## 🛠️ 工作建议

1. 新任务开始时，先看 `CODE_INDEX.md`，再深入具体文件。
2. 如果修改了核心结构、路由、类、脚本入口、部署文件，应同步运行索引更新命令。
3. 提交前若本次改动影响仓库结构，应确认 `CODE_INDEX.md` 已更新。
